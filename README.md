## Para subir a GitHub Pages

1. Crea un repo en GitHub llamado `ingcivil`
2. Sube todos los archivos de esta carpeta
3. Ve a Settings → Pages → Source: `main` branch, `/ (root)`
4. Tu app estará disponible en `https://tu-usuario.github.io/ingcivil/`
